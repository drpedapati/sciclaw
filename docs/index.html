You describe what you need in plain language. sciClaw plans, executes, and reports back.

You ask, it searches

“Find recent papers on TDP-43 in ALS.” sciClaw queries PubMed, filters by relevance, pulls full citations, and saves everything to your workspace—no tab-switching, no copy-paste.

You teach, it learns

Drop in a style guide, paste a lab protocol URL, or point it at your last three papers. sciClaw turns these into skills—reusable instructions it follows every time. Your methods, your voice, your standards—encoded once, applied consistently.

You specify, it drafts

“Draft the methods: Spearman correlations between P300 amplitude at Pz and ALSFRS-R decline, BH-corrected at FDR 0.05.” sciClaw follows your statistical plan to the letter, produces a Word doc with tracked changes, and waits for your review.

You decide, it records

Every search, draft, and revision is logged with provenance metadata. Six months later you can retrace exactly which analysis produced which result and why.

You stay in the loop. sciClaw does the legwork.

Tools built for how scientists actually work

These ship with sciClaw. The agent calls them automatically when your request needs them.

Microsoft Word Review

Real tracked changes & comments in .docx

Ask sciClaw to review your manuscript and it returns a .docx you open in Word—with tracked changes you can accept or reject and comments anchored to specific paragraphs. Exactly like getting a revision back from a co-author.

  • Tracked insertions, deletions, and replacements you review in Word
  • Margin comments tied to specific text, not just line numbers
  • Compare two drafts side-by-side with semantic diff
  • NIH-format Word template included for grant applications

PubMed Search

Full NCBI E-Utilities access

Tell sciClaw what you’re looking for and it builds the PubMed query—Boolean operators, MeSH terms, date ranges, publication types. Results export directly to Zotero, EndNote, or Mendeley.

  • Complex queries with MeSH terms, field tags, and filters
  • Walk citation chains forward (cited-by) and backward (references)
  • Find related articles scored by relevance
  • Export as RIS for one-click import into your reference manager
  • Works with your own NCBI API key for faster, higher-volume searches

Get started in four steps

Works on macOS and Linux. No coding required.

1

Install

One command installs sciClaw and all companion tools (ripgrep, pandoc, Quarto, IRL, docx-review, pubmed-cli). Pre-compiled binaries — no build step.

brew tap drpedapati/tap && brew install sciclaw

No Homebrew? Download binaries from the releases page. macOS: also install Quarto with brew install --cask quarto.

2

Set up your workspace

Creates your ~/sciclaw folder with skills, templates, and configuration. The interactive wizard walks you through authentication and optional tools.

sciclaw onboard

Use sciclaw onboard --yes to skip the wizard and accept defaults.

3

Log in with your ChatGPT account

Sign in with the same OpenAI account you use for ChatGPT. Opens a device code flow — no API key needed.

sciclaw auth login --provider openai

Or use any supported provider: Anthropic, Google Gemini, OpenRouter, Azure, DeepSeek, Groq. See authentication docs.

4

Connect a chat app

Talk to sciClaw from Telegram, Discord, or Slack. The setup wizard guides you through creating a bot and linking it.

sciclaw channels setup telegram

Start the gateway: sciclaw service install && sciclaw service start (runs in background via launchd/systemd). Or sciclaw gateway in a terminal. CLI also works: sciclaw agent -m "your question".

Works with your preferred AI

Switch between providers and models anytime. You bring your own API key.

OpenAI Anthropic Google Gemini OpenRouter DeepSeek Groq

Switch models: sciclaw models set anthropic/claude-opus-4-6

Frequently asked questions

Do I need to know how to code?

No. You chat with sciClaw through Telegram, Discord, or Slack — the same apps you already use. No terminal required. A CLI is available for power users.

Is my data sent to the cloud?

sciClaw runs on your computer. Your conversations are sent to whichever AI provider you choose (OpenAI, Anthropic, etc.) using your own API key. Nothing is shared with us.

Can I trust AI-generated content for research?

You do what you’ve always done: read critically, push back, and iterate—the same loop you’d run with a human research assistant. The difference is that sciClaw logs every source, decision, and revision, so you can retrace the reasoning anytime. If something looks off, just ask—it’s a conversation.

What operating systems are supported?

macOS (Apple Silicon and Intel), Linux (amd64, arm64, riscv64), and Windows (amd64). Pre-built binaries are available on the releases page.

Is it free?

Yes. sciClaw is MIT licensed and completely free. The only cost is your AI provider's API usage, which you control directly.
