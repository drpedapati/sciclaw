One folder. Everything in it.

Drafts, session logs, skills, audit records — all in ~/sciclaw. Open it in Finder, back it up with Git, or hand it to a collaborator.

~/sciclaw/
├── sessions/       # Every conversation, searchable
├── memory/         # Long-term notes across sessions
├── skills/         # 13 built-in + your custom skills
│   ├── scientific-writing/
│   ├── pubmed-cli/
│   ├── docx-review/
│   └── … 10 more
├── hooks/          # Event audit log (JSONL)
├── cron/           # Scheduled jobs
├── state/          # Runtime state
├── AGENTS.md       # Agent behavior rules
├── HOOKS.md        # Lifecycle hook instructions
├── IDENTITY.md     # Lab identity & voice
├── MEMORY.md       # Persistent memory template
├── SOUL.md         # Guardrails & ethics
├── TOOLS.md        # Tool usage guidelines
└── USER.md         # Your preferences

Reproducible by default

Every tool call, every LLM response, every decision is logged with timestamps. Six months from now, you can trace exactly which analysis produced which result.

Portable and auditable

The workspace is plain files — Markdown, JSON, JSONL. No database, no proprietary format. Copy it, git init it, share it with reviewers.

You control the agent

Edit AGENTS.md to change behavior, SOUL.md to set guardrails, USER.md to teach it your preferences. Plain text, version-controlled.
